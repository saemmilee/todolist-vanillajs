# ✔TO DO LIST✔


### 프로젝트 소개
------------
TO DO LIST를 구현했습니다. 
당신이 해야할 일을 등록하고, 완료하고, 제거할 수 있습니다.   

### 기능 설명
------------
HTML5에서 추가된 localstorage를 이용해 데이터를 저장하여 사용자가 지우지 않는 이상 계속 브라우저에 남아 있도록 구현했습니다.
